# Hotel-Room-Booking-System
A full-stack hotel room booking system where users can explore rooms, check availability, and book stays. This project is built to understand real-world features like authentication, CRUD operations, and booking management.

Features
User Features

User signup/login

Browse rooms with details & pricing

Check availability

Book rooms

View & manage bookings

Admin Features

Add, update & delete rooms

Manage all bookings

View users & reservations

General

Secure authentication (JWT)

Responsive UI

REST API backend
